Feature Reduction Workflow Example¶

This is an example of a feature reduction workflow, from data import to EDA, data imputation, feature reduction using 3 different methods, selection of relevant features, and price prediction on a house pricing dataset.

############## GET TOP FEATURES ##############

features = np.sum([lcv_mask, rf_mask, gb_mask], axis=0)
lm = LinearRegression()
scaler = StandardScaler()

# Create a mask for features selected by all 3 models
meta_mask = features >= 1

# Apply the dimensionality reduction on X
X_reduced = X.loc[:, meta_mask]

# Plug the reduced dataset into a linear regression pipeline
X_train, X_test, y_train, y_test = train_test_split(X_reduced, y, test_size=0.3, random_state=0)

lm.fit(scaler.fit_transform(X_train), y_train)
